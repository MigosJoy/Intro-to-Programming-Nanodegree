Lesson One: Introduction to the Web

Components of the Web

The web is made up of several components: Browers, Servers, the Internet, and a protocol called HTTP. Browers make requests via the internet to servers using HTTP (Hypertext Transfer Protocol). Servers respond with files that the browser then displays. Servers are essentially just computers that host files rather than a traditional consumer laptop.

HTML: Hyper Text Markup Language

Hyper Text Markup Language ("HTML") is made up of Text Content (what you see), Markup (what it looks like), references to other documents, and links to other pages. Most HTML Markup will typically include opening and closing tags that surround the contents. The entire line is known as the element. Certain aspects of Markup can give the contents "Attributes." For instance the anchor attribute creates links with whatever contents it is attributed to. Similarly, there is also an image attribute that displays the url to a picture for the user to see.

Whitespace

When writing HTML code it may become necessary at times to break up ideas. By default any whitespace written in the code is treated as only one space. Creating breaks can be done with two types of code: Block and/or Inline. Block breaks create a "block" around its contents. In comparison Inline creates a new line within the code while still being written on the same line as that code. An example of a Block tag would be Paragraph, which was used to create this webpage. An example of an Inline tag would be Break. This Video does an excellent job at explaining. A key difference between paragraph and break is that break is known as "Void" while paragraph is not. A Void tag does not need a closing tag.

Visual Styling

HTML and CSS are both programming languages. HTML represents the structure to Front-End Development, while CSS represents the style. CSS stands for Cascading Style Sheets. Each HTML Tag creates an element in the DOM (Document Object Model) that the Browser uses to display the page. The DOM is the tree-like structure of the webpage.
